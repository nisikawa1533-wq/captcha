API ドキュメント
日本の数学問題データベースと AI 突破困難 CAPTCHA を REST API で提供します。フリープランは登録のみで即日利用可能。
⚡ クイックスタート APIキーを取得
⚡ クイックスタート

3ステップで API を使い始めることができます。

① APIキーを取得する

sugaku-club.com/signup で登録すると、メールアドレスに APIキーが送られます。フリープランはクレジットカード不要です。

② リクエストを送る
# ランダムな数学問題を1件取得
curl https://api.sugaku-club.com/v1/problems/random \
  -H "X-API-Key: SGK-XXXX-XXXXXXXX"
const response = await fetch('https://api.sugaku-club.com/v1/problems/random', {
  headers: { 'X-API-Key': 'SGK-XXXX-XXXXXXXX' }
});
const problem = await response.json();
console.log(problem.question); // "∫₀¹ x² dx を求めよ"
import requests

res = requests.get(
    'https://api.sugaku-club.com/v1/problems/random',
    headers={'X-API-Key': 'SGK-XXXX-XXXXXXXX'}
)
print(res.json()['question'])

🔑 認証

すべてのリクエストに X-API-Key ヘッダーを付けてください。

# ヘッダーで渡す（推奨）
X-API-Key: SGK-XXXX-XXXXXXXX

# クエリパラメータでも可（非推奨・URLにキーが残るため）
GET /v1/problems/random?api_key=SGK-XXXX-XXXXXXXX
⚠️
APIキーはクライアントサイドのコード（JavaScript など）に直接埋め込まないでください。環境変数またはバックエンドで管理してください。

📐 数学問題 API
GET /v1/problems/random ランダムな問題を1件取得

条件を指定してランダムな問題を取得します。指定なしの場合は全問題からランダムに返します。

パラメータ型説明
subject任意string分野。algebra / geometry / calculus / statistics / number_theory
difficulty任意integer難易度 1〜5（1=中学基礎、5=大学受験難関）
grade任意string学年。middle / high_school / university_exam
include_solution任意boolean解説を含むか（デフォルト: true）
 
curl "https://api.sugaku-club.com/v1/problems/random?subject=calculus&difficulty=3" \
  -H "X-API-Key: SGK-XXXX-XXXXXXXX"
const params = new URLSearchParams({ subject: 'calculus', difficulty: 3 });
const res = await fetch(`https://api.sugaku-club.com/v1/problems/random?${params}`, {
  headers: { 'X-API-Key': 'SGK-XXXX-XXXXXXXX' }
});
res = requests.get('https://api.sugaku-club.com/v1/problems/random',
    params={'subject':'calculus','difficulty':3},
    headers={'X-API-Key':'SGK-XXXX-XXXXXXXX'})

🛡 CAPTCHA API
POST /v1/captcha/generate チャレンジを生成

AIが苦手とする推論型チャレンジを生成します。返却された challenge_id を使って答え合わせを行います。

ボディ型説明
locale任意string言語（ja / en。デフォルト: ja）
difficulty任意integer難易度 1〜5（デフォルト: ランダム）
category任意stringカテゴリ（physics / logic / social / math / common_sense）
expires_in任意integer有効秒数（デフォルト: 300）
 
curl -X POST https://api.sugaku-club.com/v1/captcha/generate \
  -H "X-API-Key: SGK-XXXX-XXXXXXXX" \
  -H "Content-Type: application/json" \
  -d '{"locale":"ja","difficulty":4}'
const challenge = await fetch('https://api.sugaku-club.com/v1/captcha/generate', {
  method: 'POST',
  headers: { 'X-API-Key': 'SGK-XXXX-XXXXXXXX', 'Content-Type': 'application/json' },
  body: JSON.stringify({ locale: 'ja', difficulty: 4 })
}).then(r => r.json());
{ "challenge_id": "chg_3mPqR7nXv", "category": "physics", "question": "足の上に落としたとき最も痛いのはどれ？", "options": ["鉄塊", "木材", "綿", "シャボン玉"], "expires_at": "2026-06-13T10:00:00+09:00" }
⚠️
正解（answer）はレスポンスに含まれません。クライアントが答えを送信したあと /captcha/verify でサーバーサイド検証してください。
POST /v1/captcha/verify 回答を検証してトークンを発行
ボディ型説明
challenge_id必須stringgenerateで取得したチャレンジID
answer必須integer選択肢のインデックス（0始まり）
 
curl -X POST https://api.sugaku-club.com/v1/captcha/verify \
  -H "X-API-Key: SGK-XXXX-XXXXXXXX" \
  -H "Content-Type: application/json" \
  -d '{"challenge_id":"chg_3mPqR7nXv","answer":0}'
// 正解の場合 { "verified": true, "token": "vt_9xKjQ2mN8pLr...", // フォーム送信時に付与して再検証に使う "expires_at": "2026-06-13T10:15:00+09:00" } // 不正解の場合 { "verified": false, "token": null }
✅
発行された token をフォームに hidden field として含め、バックエンドで再度 verify することでスパム送信を防げます。

📣 Webhook

スタンダードプラン以上で、月間リクエスト数が指定の閾値に達したときに Webhook で通知を受け取れます。

Webhook ペイロード例
{ "event": "usage.threshold_reached", "threshold_percent": 80, "used": 160000, "limit": 200000, "timestamp": "2026-06-13T09:31:00+09:00" }
ℹ️
Webhook URL の登録はダッシュボードから。HMAC-SHA256 で署名した X-Sugaku-Signature ヘッダーでリクエストの正当性を検証できます。
