はじめに

sugaku-club.com API を使うと、AI耐性CAPTCHAの発行・検証、数学問題の取得・採点、トランザクションメールの送信をシンプルなREST APIで利用できます。

application/json

全てのリクエストはHTTPSで行い、レスポンスは application/json です。エラー時も同様にJSONが返ります。

認証

まずアカウントを作成して APIキー を取得し、以後の全リクエストに X-Api-Key ヘッダーとして付与します。

POST /api/auth/signup

新規アカウントを作成し、JWTトークンとAPIキーを返します。

# リクエスト POST https://sugaku-club.com/api/auth/signup Content-Type: application/json { "email": "user@example.com", "password": "password123" } # レスポンス 201 { "token": "eyJ...", "api_key": "sk_XXXX" }
POST /api/auth/login

既存アカウントでログインし、JWTトークンとAPIキーを再取得します。

# リクエスト POST https://sugaku-club.com/api/auth/login Content-Type: application/json { "email": "user@example.com", "password": "password123" } # レスポンス 200 { "token": "eyJ...", "api_key": "sk_XXXX" }

CAPTCHA API

AI耐性の数学的CAPTCHAを発行・検証します。チャレンジを取得してユーザーに解かせ、回答を送信するとトークンが発行されます。

POST /api/captcha/issue

新しいCAPTCHAチャレンジを発行します。有効期限は発行から5分間です。

# リクエスト POST https://sugaku-club.com/api/captcha/issue X-Api-Key: sk_XXXX # レスポンス 200 { "challenge_id": "ch_XXXX", "challenge": "...", "expires_at": 1750000000 }
POST /api/captcha/verify

ユーザーの回答を検証します。正答であれば使い捨てトークンが返ります。

# リクエスト POST https://sugaku-club.com/api/captcha/verify X-Api-Key: sk_XXXX Content-Type: application/json { "challenge_id": "ch_XXXX", "answer": "答え" } # レスポンス 200 { "valid": true, "token": "SGK-XXXX" }
GET /api/captcha/challenge

チャレンジ問題（表示用データ）を取得します。

# リクエスト GET https://sugaku-club.com/api/captcha/challenge X-Api-Key: sk_XXXX # レスポンス 200 { "challenge_id": "ch_XXXX", "challenge": "次の計算の答えは？ 12 × 7 − 4", "type": "arithmetic", "expires_at": 1750000000 }

数学問題 API

難易度別の数学問題を取得・採点できます。学習アプリや練習ツールへの組み込みに最適です。

GET /api/math/problem

クエリパラメータ level で難易度（1〜5）を指定して問題を取得します。

# リクエスト GET https://sugaku-club.com/api/math/problem?level=1 X-Api-Key: sk_XXXX # レスポンス 200 { "id": "p_XXXX", "question": "3 + 5 = ?", "level": 1 }
POST /api/math/check

問題IDと回答を送信し、正誤判定と解説を受け取ります。

# リクエスト POST https://sugaku-club.com/api/math/check X-Api-Key: sk_XXXX Content-Type: application/json { "id": "p_XXXX", "answer": 8 } # レスポンス 200 { "correct": true, "explanation": "3+5=8" }

メール送信 API NEW

トランザクションメールをシンプルなAPIで送信できます。フォーム送信通知、会員登録確認、パスワードリセットなどに活用できます。

POST /api/mail/send

宛先・件名・本文を指定してメールを送信します。送信後に月間使用数と残り数を返します。

# リクエスト POST https://sugaku-club.com/api/mail/send X-Api-Key: sk_XXXX Content-Type: application/json { "to": "user@example.com", "subject": "件名", "text": "本文" } # レスポンス 200 { "success": true, "used": 1, "remaining": 99 }
GET /api/mail/quota

現在の月間メール送信クォータと使用状況を返します。

# リクエスト GET https://sugaku-club.com/api/mail/quota X-Api-Key: sk_XXXX # レスポンス 200 { "plan": "free", "quota_per_month": 100, "used": 1, "remaining": 99 }

エラーコード

全てのエラーレスポンスは {"error":"...","code":...} 形式のJSONで返ります。

HTTPステータス 意味 対処
400リクエスト不正パラメータの型・必須フィールドを確認
401APIキー不正X-Api-Key ヘッダーとキーの値を確認
404リソースが見つからないID（challenge_id / id）が正しいか確認
422回答期限切れ新しいチャレンジを再発行してください
429レート制限超過しばらく待つかプランをアップグレード
503サービス一時停止ステータスページを確認してください
# エラーレスポンス例 (401) { "error": "Invalid API key", "code": 401 }
