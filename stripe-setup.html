💳 Stripe でカード決済を有効にする

口座なし・0円で開設できます。売上が出たら親の口座に振り込めばOK。
月3件の有料プランで VPS 代が出ます。

⚠️ 未成年の場合: Stripe の登録は親権者名義で行ってください。親御さんに頼んで一緒に設定しましょう。
STEP 1

Stripeアカウントを作る 無料

以下のリンクからアカウントを作成します（手数料は売上から自動差引、月額費用なし）

Stripe に登録する（無料）→
  • メールアドレス・パスワードで登録
  • 本人確認は親権者の情報で入力
  • 銀行口座は親の口座を登録（売上がここに振り込まれる）
STEP 2

シークレットキーを取得する

Stripeダッシュボードで:

  • 左メニュー「開発者」→「APIキー」
  • 「シークレットキー」の「表示」ボタン
  • sk_live_XXXX... をコピー

⚠️ シークレットキーは絶対に公開しないこと

STEP 3

サーバーに設定する（SSH 2コマンドだけ）

Oracle Cloud サーバーに SSH ログインして実行:

pm2 set sugaku-user-api:STRIPE_SECRET_KEY sk_live_ここにキーを貼る
pm2 restart sugaku-user-api

✓ これだけでカード決済が有効になります

STEP 4

Webhook を設定する（自動でプラン反映）

Stripeダッシュボード →「開発者」→「Webhook」→「エンドポイントを追加」

https://sugaku-club.com/api/billing/webhook
  • イベント: checkout.session.completed にチェック
  • 「署名シークレット」をコピーして:
pm2 set sugaku-user-api:STRIPE_WEBHOOK_SECRET whsec_XXXX
