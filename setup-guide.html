メール・決済の設定ガイド

Googleアカウントと PayPay があれば親なし・口座なし・無料で始められます。

📧 メール送信を有効にする（Gmail） 13歳〜OK 無料
Googleアカウントがあれば即設定できます。Oracle Cloudのポート25制限も関係ありません（ポート587を使用）。
専用Gmailを作る
サービス用のGmailアドレスを1つ作成（例: sugakuclub.mail@gmail.com）
→ Googleアカウント作成
2段階認証をONにする
myaccount.google.com → セキュリティ → 2段階認証プロセス → 有効にする
アプリパスワードを発行する
myaccount.google.com → セキュリティ → アプリパスワード
「アプリを選択」→「その他」→「sugaku-api」などと入力 → 16文字のパスワードをコピー
サーバーに設定する（SSH）
pm2 set sugaku-user-api:MAIL_TRANSPORT gmail
pm2 set sugaku-user-api:GMAIL_USER あなたのアドレス@gmail.com
pm2 set sugaku-user-api:GMAIL_PASS "16文字のアプリパスワード"
pm2 restart sugaku-user-api
✓ 設定後: 振込申請メール・プラン変更メールが自動で届くようになります。メール送信APIも使えます（フリープラン: 100通/月）
💚 PayPay で料金を受け取る 13歳〜OK・口座不要 手数料0円
PayPayは個人間送金なら手数料0円・口座不要・中学生でもOK。
ユーザーが送金 → PayPay残高に貯まる → コンビニで使う or 将来口座に出金
PayPayアプリをインストール
App Store / Google Play → 「PayPay」
電話番号（自分のスマホ）で登録できます
自分のPayPay IDを確認する
PayPayアプリ → ホーム → 右上アイコン → PayPay ID（@から始まる文字列）
サーバーに設定する（SSH）
pm2 set sugaku-user-api:PAYPAY_ID @あなたのPayPayID
pm2 restart sugaku-user-api
✓ 設定後: ユーザーが「PayPayで支払う」を選ぶと、あなたのPayPay IDとメモ欄の参照番号が表示されます。
⚠️ 受け取り確認: PayPayで入金を確認したら admin.html の「入金管理」で「確認済み」にしてください。自動でプランが変わります。
