1適用範囲・定義

本規約は、sugaku-club.com が提供する数学学習サービス・API（以下総称して「本サービス」）の利用条件を定めるものです。

  • 「ユーザー」とは、本サービスに登録した個人・法人を指します。
  • 「コンテンツ」とは、問題集・解説・フォーラム投稿・APIレスポンスを含む一切の情報を指します。
  • 「APIキー」とは、本サービスのAPIにアクセスするための認証情報を指します。

2アカウント登録

本サービスへの登録は、以下の条件を満たす方に限り可能です。

  • 13歳以上であること（13歳未満は保護者の同意が必要）
  • 正確かつ最新の情報を登録すること
  • 1人につき1アカウントのみ保持すること
  • アカウントの認証情報を第三者と共有しないこと

虚偽の情報による登録、または規約に違反した場合、予告なくアカウントを停止・削除することがあります。

3サービスの利用

本サービスは教育・学習目的での利用を前提としています。ユーザーは本規約および関係法令を遵守した上でサービスをご利用ください。

  • フォーラムへの投稿は自己責任で行い、他ユーザーへの尊重を忘れないこと
  • 本サービスのコンテンツを無断で転載・再配布しないこと
  • 自動化ツール（ボット等）による大量アクセスは API プランの範囲内でのみ許可

4APIの利用

APIキーはご自身のアプリケーション・サービスでのみ使用できます。

  • APIキーの第三者への譲渡・共有・公開は禁止します
  • 各プランに定めるレート制限・月間リクエスト上限を遵守すること
  • APIを利用したサービスには「Powered by sugaku-club.com API」の表記を推奨します
  • 従量課金プランでは、月末に実際のリクエスト数に基づいた請求が発生します
  • 超過リクエストは自動的に課金対象となります（上限設定が可能）

🔑 APIキーの不正利用が発覚した場合、即時無効化および法的措置を取る場合があります。

5料金・支払い

定量課金プランは月額前払い、従量課金は月末締め翌月請求です。

  • 料金は消費税込みで表示されます
  • 支払い方法：クレジットカード（Visa/Mastercard）・銀行振込
  • 月途中での解約の場合、当月分の返金は行いません
  • 定量課金プランの年払いは、未使用月分の返金はいたしかねます
  • 従量課金の月間上限アラートを設定した場合、上限到達時に自動停止します

料金改定の際は、30日前までにメールにてお知らせします。

6禁止事項

以下の行為を明示的に禁止します。

  1. 本サービスへの不正アクセス・サーバー攻撃（DDoS等）
  2. 他ユーザーへの嫌がらせ・誹謗中傷・スパム投稿
  3. 著作権・商標権等の知的財産権を侵害するコンテンツの投稿
  4. 虚偽の情報によるアカウント作成
  5. APIを利用した違法サービスの構築
  6. CAPTCHAの機械的突破の試み（研究目的を除く）
  7. 本サービスのリバースエンジニアリング

7知的財産権

本サービス上のコンテンツ（問題・解説・デザイン・ソースコード等）の著作権は sugaku-club.com または正当な権利者に帰属します。

ユーザーがフォーラムに投稿したコンテンツの著作権はユーザーに帰属しますが、本サービスの運営・改善目的での利用を許諾するものとします。

8免責事項

本サービスは現状有姿で提供されます。以下の事項について当方は責任を負いません。

  • サービスの中断・障害によるデータの損失
  • ユーザー間のトラブル
  • 本サービスの利用により生じた間接的損害
  • 第三者のウェブサイトへのリンク先のコンテンツ

当方の責任は、ユーザーが直近3ヶ月間に支払った料金を上限とします。

9サービスの変更・終了

当方はサービスの内容を予告なく変更・追加・削除することがあります。サービス終了の場合は、60日前までにユーザーに通知します。

10準拠法・管轄裁判所

本規約は日本法に準拠します。本サービスに関する紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。
